# ChangeNick
ChangeNick plugin to PocketMine-MP

##Commands
|Command|SubCommand|Info|
|------|------|-----|
|/setplayernick| [ displaynickname ]|edit your display nick name!!!|

##Features
- Set your Language in Config!
- Create your config language! example: pt_br.yml
